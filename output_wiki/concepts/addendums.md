# Addendums

The **Addendums** section serves as a repository for supplementary information and specific notes that complement the main body of the CUDA C Programming Guide. This section is designed to capture detailed behavioral specifications and limitations that may not fit within the primary instructional flow but are critical for advanced usage.

A primary example of content found in this section is the documentation of specific behaviors and limitations associated with the **CUDA stream-ordered memory allocator** [CUDA_C_Programming_Guide:L15862-L15863]. These notes ensure that developers are aware of edge cases and implementation details that affect memory management in stream-ordered contexts.
