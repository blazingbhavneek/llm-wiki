# CDP1 Execution Environment

The execution environment and memory model for Compute Device Program 1 (CDP1) are defined within the broader context of the CUDA C programming guide. For the specific implementation details regarding CDP1, the documentation refers readers to the general "Execution Environment and Memory Model" section.

## Relation to CDP2

The documentation explicitly distinguishes between CDP1 and CDP2 versions. While CDP1 relies on the standard execution environment description, a separate section covers the "Execution Environment and Memory Model" for the CDP2 version of the document [CUDA_C_Programming_Guide:L14292-L14295].
