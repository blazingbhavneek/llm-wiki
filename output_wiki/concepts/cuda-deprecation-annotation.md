# Deprecation Annotation in CUDA

The NVIDIA CUDA compiler (`nvcc`) provides support for deprecation annotations, allowing developers to mark functions, variables, or types as deprecated. The specific syntax supported depends on the host compiler being used and the C++ dialect enabled.

## Supported Attributes

`nvcc` supports the following deprecation mechanisms:

*   **Host Compilers (gcc, clang, xlC, icc, pgcc):** The `deprecated` attribute is supported.
*   **Host Compiler (cl.exe):** The `deprecated` `declspec` is supported.
*   **C++14 Dialect:** The standard `[[deprecated]]` attribute is supported when the C++14 dialect is enabled.

## Compilation Phases and Diagnostics

Deprecation diagnostics are generated based on the compilation phase in which the reference to the deprecated entity occurs:

1.  **Device Compilation:** When `__CUDA_ARCH__` is defined (indicating device code compilation), the CUDA frontend compiler generates a deprecation diagnostic for references to deprecated entities from within the body of `__device__`, `__global__`, or `__host__ __device__` functions [CUDA_C_Programming_Guide:L17455-L17461].
2.  **Host Compilation:** References to deprecated entities from within `__host__` functions (or other host-side contexts) are handled by the host compiler, which generates its own diagnostics [CUDA_C_Programming_Guide:L17455-L17461].

## Suppression of Warnings

Deprecation warnings can be suppressed using specific flags or pragmas, though behavior differs between the CUDA frontend and host compilers:

*   **Host Compiler Pragmas:** The CUDA frontend compiler does **not** support `#pragma gcc diagnostic` or `#pragma warning` mechanisms. Consequently, deprecation diagnostics generated by the CUDA frontend compiler are not affected by these host-specific pragmas [CUDA_C_Programming_Guide:L17455-L17461].
*   **NVIDIA-Specific Pragma:** To suppress warnings specifically for device code, users can use the NVIDIA-specific pragma `#pragma nv_diag_suppress` [CUDA_C_Programming_Guide:L17455-L17461].
*   **nvcc Flags:**
    *   `-Wno-deprecated-declarations`: Suppresses all deprecation warnings [CUDA_C_Programming_Guide:L17455-L17461].
    *   `-Werror=deprecated-declarations`: Turns deprecation warnings into errors [CUDA_C_Programming_Guide:L17455-L17461].
