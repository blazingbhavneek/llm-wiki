# 64-Bit Compatibility

The 64-bit version of `nvcc` compiles device code in 64-bit mode, meaning that pointers are 64-bit [CUDA_C_Programming_Guide:L1207-L1210]. Device code compiled in 64-bit mode is only supported with host code compiled in 64-bit mode [CUDA_C_Programming_Guide:L1207-L1210].

## Aliases

- 64-bit Device Code
