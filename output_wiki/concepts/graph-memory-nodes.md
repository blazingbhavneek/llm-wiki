# Graph Memory Nodes

Graph Memory Nodes are a specific type of node within the CUDA Graph execution model, allowing for memory operations to be captured and replayed as part of a graph. They are detailed in Chapter 16 of the CUDA C Programming Guide.

## Overview

The CUDA C Programming Guide provides documentation on Graph Memory Nodes in Chapter 16. This chapter covers the functionality and usage of these nodes within the broader context of CUDA Graphs, which enable efficient capture and replay of kernel launches and memory operations.
