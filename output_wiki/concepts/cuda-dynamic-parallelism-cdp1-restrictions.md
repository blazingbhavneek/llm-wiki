# CUDA Dynamic Parallelism (CDP1) Implementation Restrictions

Dynamic Parallelism guarantees all semantics described in the CUDA documentation, but certain hardware and software resources are implementation-dependent. These dependencies limit the scale, performance, and other properties of a program that uses the device runtime [CUDA_C_Programming_Guide:L14908-L14914].

For the CDP2 version of these restrictions, see the corresponding section in the CDP2 documentation [CUDA_C_Programming_Guide:L14908-L14914].
