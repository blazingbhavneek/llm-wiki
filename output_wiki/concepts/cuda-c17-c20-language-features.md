# CUDA C++17 and C++20 Language Features

NVIDIA's CUDA compiler (`nvcc`) provides support for modern C++ language standards, enabling developers to use newer language features in device code.

## C++17 Support

All C++17 language features are supported in `nvcc` version 11.0 and later. This support is subject to specific restrictions outlined in the CUDA C++ Programming Guide [CUDA_C_Programming_Guide:L16522-L16529].

## C++20 Support

All C++20 language features are supported in `nvcc` version 12.0 and later. Similar to C++17, this support is subject to restrictions described in the official documentation [CUDA_C_Programming_Guide:L16522-L16529].
